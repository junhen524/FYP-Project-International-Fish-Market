# FishMarket PHP for XAMPP

1. Put this folder inside `xampp/htdocs/`.
2. Start Apache in XAMPP.
3. Open `http://localhost/FishMarket-php-xampp/`.

Notes:
- This version removes runtime SQLite/MySQL requirements.
- Product, category, demo order, and profile data are loaded from JSON.
- Cart uses PHP session.
- Login, register, checkout, wallet top-up, AI chat, and admin actions are preserved as UI and ready for you to connect to Supabase/API later.
